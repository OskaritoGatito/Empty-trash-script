# Linux system based Auto-Trash Emptier

Automatic functionality to empty your system trash/files directory everyday.

## Description
A lightweight, secure, and systemd-driven automation tool designed for Fedora (and other systemd-based Linux distributions) to automatically clean up your trash bin on a daily basis. It runs entirely within the user space, meaning it requires **no root privileges (no sudo)** for daily operations and installation.

## Installation

To install this tool for your current user, clone this repository or download the files, open your terminal inside the project directory, and run:

```bash
chmod +x install.sh
./install.sh
